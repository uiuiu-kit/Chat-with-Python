Befehlsdokumentation

inputImg(Promt:string) -> Pillow Image

Mit dem Befehl inputImg kannst du den Nutzer deines Programmes nach einem Bild fragen. Wenn du im Chat dann ein Bild hochlädst wird dieses dann in ein Pillow Image Objekt umgewandelt was du im weiteren verlauf verwendet kannst. Mit .show() kannst du dir das Bild im Chat anzeigen lassen.

Beispielverwendung:

img = inputImg("Übergib ein Bild")  # fragt nach einem Bild
 img.show()                          # zeigt das Bild im Chat an

inputTable(Promt:string) -> pandas.dataframe

Mit dem Befehl inputTable kannst du den Nutzer deines Programmes nach einer Tabelle fragen. Wenn du im Chat dann ein Tabelle hochlädst wird dieses dann in ein pandas Dataframe Objekt umgewandelt was du im weiteren verlauf verwendet kannst. Mit print([name deines Dataframes]) kannst du dir die Tabelle im Chat anzeigen lassen.

Beispielverwendung:

df = inputTable("Übergib ein Tabelle")  # fragt nach einer Tabelle
 print(df)                               # zeigt die Tabelle im Chat an
